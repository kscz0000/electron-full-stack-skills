# Electron Skills Collection

This repository contains a curated collection of Electron-related skills from multiple sources:

## Included Skills

### 1. Electron Core Skills (from partme-ai/full-stack-skills)
- `electron` - Core Electron framework guidance
- `electron-egg` - Electron EGG framework integration

### 2. Desktop Application Development
- Comprehensive Electron desktop app development
- Main process and renderer process management
- IPC communication patterns
- Window management and native features

### 3. Build and Packaging
- Application packaging with Electron Forge
- Cross-platform distribution
- Auto-updater implementation
- Security best practices

## Usage

Clone this repository and integrate the skills into your Claude Code environment:

```bash
git clone https://github.com/kscz0000/electron-full-stack-skills.git
```
